Fundamentos de Internet

Internet es una red global de computadoras interconectadas que utilizan protocolos estandarizados para intercambiar información.

Diferencia entre Internet y la Web

Internet World Wide Web
Infraestructura global de redes. Sistema de páginas web accesibles mediante navegadores.
Incluye servicios como correo electrónico y FTP. Permite visualizar sitios web.
Funciona con TCP/IP. Funciona principalmente con HTTP y HTTPS.

Protocolos de Internet

  • HTTP — protocolo que permite transferir páginas web.
  • HTTPS — versión segura de HTTP que cifra la información.
  • FTP — utilizado para transferir archivos.
  • DNS — traduce nombres de dominio a direcciones IP.
  • TCP/IP — conjunto de protocolos base de Internet.
